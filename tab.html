Tab examples

Kitbox tab has been designed to be accessible (using ARIA technology). Inspiration has been taken from Bootstrap, Jeff Smith, Athena ICT, and W3C.

By default, no styling is applied (other than to hide inactive tab panels). However, you may apply the class "line" to the kb_tabs container to get something which looks like example 2.

Also, please note that you can nest tabs within tabs.
